METHOD 1: ENVIRONMENT VARIABLE (RECOMMENDED ✅)

In your terminal, paste:

  For Gemini:
    export GOOGLE_API_KEY="paste-your-key-here"
  
  For OpenAI:
    export OPENAI_API_KEY="paste-your-key-here"

Verify:
  python test_env_setup.py


METHOD 2: .env FILE (EASIEST)

1. Create file: .env (in project root)
2. Paste inside:
   GOOGLE_API_KEY=paste-your-key-here
   
   OR
   
   OPENAI_API_KEY=paste-your-key-here

3. Test:
   python test_env_setup.py

File: post purchase support agent/agents.py

To:
   api_key="paste-your-actual-key-here"

⚠️ REMOVE BEFORE COMMITTING TO GIT!
